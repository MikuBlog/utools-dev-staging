# utools 网页开发脚手架

## 技术栈

react + ts + antd :fire:

## 开发方式

按照react脚手架的开发模式开发即可，utools插件api请参照：https://u.tools/docs/developer/welcome.html

## 打包插件

1. 执行`npm run build`
2. 在开发者中心选择打包好的`build`目录即可
