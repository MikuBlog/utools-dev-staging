# utools 网页开发脚手架

## 技术栈

react + ts + antd :fire:

## 开发流程

1. 安装依赖：`npm i`
2. `utools`搜索`开发者工具`
3. 创建项目，并选择项目下的`public/plugin.json`
4. 启动并输入`脚手架`进行调试即可，更多配置看：https://u.tools/docs/developer/welcome.html
5. 享受你的开发之旅吧~

> 建议分离窗口，这样可以一边开发一边调试（热更新）从而提高开发效率

## 打包插件

1. 执行：`npm run build`
2. 在开发者中心选择打包好的`build`目录即可
